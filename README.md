# 🎓 АлтГТУ Тренажёр

Приложение для подготовки к вступительным испытаниям АлтГТУ (2026) в стиле Duolingo.

## Как запустить

Просто открой файл **index.html** двойным кликом — приложение работает в браузере без интернета и без установки. Прогресс сохраняется в браузере (localStorage).

## Что внутри

- **3 курса**: Математика (10 тем), Информатика (7 тем), Русский язык (9 тем) — темы соответствуют официальным программам вступительных испытаний.
- **Уроки как в Duolingo**: сначала короткая теория, потом практика. Форматы заданий: выбор ответа, ввод ответа, выбор нескольких верных, сопоставление пар.
- **Интервальные повторения (система Лейтнера)**: ошибки возвращаются через 1 → 2 → 4 → 7 → 14 дней — ровно тогда, когда мозг начинает забывать. Прошёл все интервалы — задание выучено навсегда.
- **План на сегодня**: урок (новое) + повторение (закрепление) + блиц (беглость) — три компонента эффективной тренировки на главном экране.
- **⚡ Блиц**: 60 секунд, задания вперемешку из всех предметов (интерливинг), серии, рекорды.
- **Умная тренировка**: подтягивает слабые темы и ошибки к повторению.
- **Генераторы задач**: по математике, информатике и русскому (ударения, формы слов) задания собираются случайно — можно тренироваться бесконечно.
- **Пробный экзамен**: реальные задания демовариантов АлтГТУ, таймер (180/180/100 минут), шкала 100 баллов и сравнение с проходным (40/46/40), полный разбор после завершения.
- **Геймификация**: XP, стрик 🔥 с заморозками ❄️ (каждые 5 дней стрика — заморозка в запас), уровни тем (3 звезды → 👑), 16 достижений, лучший балл за экзамен.
- **Резервная копия**: экспорт и импорт прогресса файлом (Настройки → Резервная копия) — удобно переносить между браузерами.

В основе — три техники с самой сильной доказательной базой: активное воспроизведение (retrieval practice), интервальные повторения (spacing) и чередование тем (interleaving).

## Файлы

| Файл | Что это |
|---|---|
| `index.html` | точка входа |
| `style.css` | оформление |
| `app.js` | движок (уроки, экзамен, прогресс) |
| `data_math.js` | курс математики + генераторы |
| `data_inf.js` | курс информатики + генераторы |
| `data_rus.js` | курс русского языка |

Чтобы добавить задание — допиши объект в `pool` нужной темы в файле данных.
